生命中探险的区域

爱情，它在我们的青春期出现，就像在一个奇异的早晨，我听见了一阵陌生的敲门声，由此我打开了门。门外站着的货郎，脸上堆满了微笑，一定要我买下他手中的那只万花筒。我禁不住诱惑，便买下了那只神奇的万花筒。是的，它的确是新奇的，只要我用手晃动它，就会带来斑斓的世界，里面有超越我们现实的，像星空一样邈远的星际，有亮晶晶的泉水……爱情从一开始就是我们禁不住诱惑而探险的区域之一，从一开始它就是一个无法讲清楚的问题和陷阱，在我生活的那种小县城，当我正值青春年华时，我和同时代的少男少女们经历着爱情这个词汇的熔炼。它也许是突如其来的，年轻时的爱情更多的是飞蛾扑火……

爱情，来自男女的区域。这是一个广大而窄小的地区，有时候爱情会从沙漠进入绿洲，有时候爱情又会从绿洲走入沙漠——那种闪电似的灼热迅速冷却成冰。

爱情的关系，是另外一个社会意识形态，它里面有王和奴仆，也有信仰和数字。所以我以为爱情是一种探险的区域。
